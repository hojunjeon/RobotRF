Robot Sorting RL MVP Architecture

WSL2 Runtime

Ubuntu 22.04, Python venv, PyTorch, MuJoCo, Gymnasium-Robotics, Stable-Baselines3.

GoalEnv

Dict observation with observation, achieved_goal, desired_goal, and vectorized sparse reward.

SAC + HER

MultiInputPolicy with HerReplayBuffer. Other algorithms stay as extension candidates.

Portfolio Outputs

Checkpoints, TensorBoard curves, 100-episode success rate, and before/middle/after videos.

Stage 1

One object, one box. Learn basic pick and place behavior in the tabletop workspace.

Stage 2

One object, three boxes. Object type is provided in state and selects the target box.
